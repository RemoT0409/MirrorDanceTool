# 🎬 舞蹈片段學習工具

一個專為舞蹈學習者設計的工具，可以從 YouTube 影片中拆解片段，反覆練習每個動作。

## ✨ 核心功能

### 1️⃣ 貼影片連結（支援 YouTube / Bilibili）
- 直接貼上 YouTube 或 Bilibili 影片 URL
- 自動嵌入播放器（iframe）
- 不儲存影片，不觸碰版權問題
- 立即開始使用

### 2️⃣ 自動分段（新功能！）
- 設定每段時間長度（例如：每 10 秒一段）
- 一鍵自動將整部影片切分成多個段落
- 自動分段會顯示在頂部，可直接點擊跳轉到第 N 段
- 快速瀏覽和定位影片內容

### 3️⃣ A–B 手動段落切割
- 手動設定開始/結束時間
- 一鍵「新增練習段」
- 支援時間格式：`MM:SS.m`（例如：`00:42.5` → `00:48.0`）
- 學舞的本質就是反覆練「一小段」
- 自動分段和手動段落會合併顯示在練習清單中

### 4️⃣ 段落 Loop 播放
- 每一段都可以無限循環
- 單段播放功能
- 切段後自動 loop
- 循環播放時有視覺指示器

### 5️⃣ 每段「文字註解」
- 可輸入重心、拍子、技巧提醒
- 例如：「這裡手會慢一點」
- 這是與 YouTube 最大的差異點

### 6️⃣ 練習清單（列表視角）
- 顯示段落名稱
- 顯示時間區間
- 顯示備註
- 點擊即跳到該段
- 把「時間軸」變成「結構」

## 🚀 使用方法

1. **開啟網頁**
   - 直接用瀏覽器開啟 `index.html`

2. **載入影片**
   - 在輸入框貼上 YouTube 或 Bilibili 連結
   - 點擊「載入影片」按鈕

3. **自動分段（可選）**
   - 設定每段時間長度（例如：10 秒）
   - 點擊「自動分段」按鈕
   - 影片會自動切分成多個段落
   - 在預覽區點擊任意段落即可跳轉

4. **手動新增段落**
   - 播放影片到想要的開始時間
   - 點擊「設為當前時間」設定開始時間
   - 播放到結束時間
   - 點擊「設為當前時間」設定結束時間
   - （選填）輸入段落名稱
   - 點擊「新增練習段」

5. **練習段落**
   - 在練習清單中點擊「播放」單次播放
   - 點擊「循環播放」無限循環該段落
   - 點擊「編輯」可修改段落資訊和添加註解

6. **編輯註解**
   - 點擊段落的「編輯」按鈕
   - 在註解欄位輸入：
     - 重心提示
     - 拍子說明
     - 技巧提醒
   - 點擊「儲存」

## 💾 資料儲存

所有段落資料會自動儲存在瀏覽器的 `localStorage` 中，即使關閉瀏覽器也不會遺失。

## 🎨 特色

- **現代化 UI**：美觀的漸層設計，響應式佈局
- **直覺操作**：簡單易懂的介面
- **即時反饋**：視覺化的循環指示器
- **資料持久化**：自動儲存，不用擔心資料遺失

## 📱 響應式設計

支援桌面和行動裝置，隨時隨地都能練習！

## 🔧 技術細節

- 純前端實現（HTML + CSS + JavaScript）
- 使用 YouTube IFrame API
- localStorage 資料儲存
- 無需後端伺服器

## 📝 注意事項

- 需要網路連線（載入影片）
- 建議使用現代瀏覽器（Chrome、Firefox、Edge、Safari）
- YouTube 影片必須允許嵌入播放
- **自動分段功能目前僅支援 YouTube 影片**（Bilibili 由於 API 限制，部分功能可能受限）
- Bilibili 影片的時間控制功能較為有限，建議使用 YouTube 以獲得完整體驗
